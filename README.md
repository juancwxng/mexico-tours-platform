# Costa Franca Tours 🌴

A modern, fast, and scalable platform for tourism in Mazatlán. This project represents the complete modernization and rebranding of a legacy tourism platform (formerly Central Tours), delivering a premium user experience, superior performance, and a robust technical architecture.

Developed by **Juan Wong**.

## 🚀 Tech Stack

This project is built with a modern web development stack focused on performance, maintainability, and a high-end UI:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [Radix UI](https://www.radix-ui.com/) (Primitives) + `clsx` & `tailwind-merge`
* **Icons:** [Lucide React](https://lucide.dev/)
* **Typography & Design:** Optimized with `@tailwindcss/typography` for a clean, minimalist, and professional aesthetic.

## 📦 Project Structure

The application utilizes the Next.js App Router architecture (without a `src` directory), configured with import aliases (`@/*`) for cleaner and more maintainable code.

Key pre-configured UI dependencies include:
* `@radix-ui/react-dialog`
* `@radix-ui/react-slot`
* `class-variance-authority`
* `tailwindcss-animate`

## 💻 Local Development

Follow these steps to set up the development environment on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd costa-franca-tours
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the application:**
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 🛠 Useful Commands

* `npm run build` - Builds the application for production.
* `npm run start` - Starts the production server.
* `npm run lint` - Runs ESLint to catch and fix code issues.
